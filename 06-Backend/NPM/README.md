# BIENVENIDOS
## Calculadora Udla

este paquete nos ayuda a sumar, restar y multiplicar.

```javascript
calculadoraUdla = {
  sumar:(n1,n2)=>{
      return n1+n2;
    },
    restar:(n1,n2)=>{
      return n1-n2;
    },
    multiplicar:(n1,n2)=>{
      return n1*n2;
    }
};

```
